# Shopify Camera Theme Starter

Minimal Online Store 2.0 Shopify theme starter with a bold, modern look.

## Folder structure

- `layout/`
- `templates/`
- `sections/`
- `assets/`
- `config/`
- `locales/`

## Import into Shopify

1. In Shopify admin, go to **Online Store > Themes**.
2. Click **Add theme > Upload zip file**.
3. Upload a zip generated from this folder.

## Create zip from terminal

Run this in the parent directory of this project:

```bash
cd /Users/leticiacoelho/Desktop
zip -r shopify-camera-theme.zip shopify-camera
```

Then upload `shopify-camera-theme.zip` in Shopify.
